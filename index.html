Qué es un DHCP

El protocolo de configuración dinámica de host (DHCP, Dynamic Host Configuration Protocol),es un estándar TCP/IP que simplifica la administración de la configuración IP haciéndola automática. El servidor DHCP recibe peticiones de clientes solicitando una configuración de red IP. Responde proporcionando los parámetros que permitan a los clientes autoconfigurarse. Los clientes hay que configurarlo seleccionando la opción “Obtener dirección IP automáticamente”.

Conceptos DHCP

Ámbito servidor DHCP: agrupamiento de equipos.
Rango: direcciones que se pueden asignar.
Ejemplo: 192.168.0.1 a 192.168.0.254
