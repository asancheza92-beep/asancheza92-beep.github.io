HTML, siglas de HyperText Markup Language (Lenguaje de Marcas de Hipertexto), es el lenguaje estándar utilizado para crear y estructurar páginas web. Se trata de un lenguaje de marcado compuesto por etiquetas que definen la estructura y el contenido de una página, como texto, imágenes, enlaces, vídeos y otros elementos multimedia. Estas etiquetas, encerradas entre corchetes angulares, indican al navegador cómo interpretar y presentar el contenido, actuando como el esqueleto de cualquier sitio web. HTML proporciona la base estructural sobre la cual se construyen las páginas web, permitiendo que los navegadores muestren la información de forma organizada y accesible.

CSS, o Hojas de Estilo en Cascada, es un lenguaje de estilos que define la apariencia de los documentos HTML, como el color, la fuente, el tamaño, la posición y el diseño de una página web. Separa la estructura del contenido (HTML) de su presentación, lo que permite a los desarrolladores y diseñadores crear sitios web visualmente atractivos y consistentes, facilitando el mantenimiento y la flexibilidad del diseño.
